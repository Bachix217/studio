# **App Name**: Tacoto.ch

## Core Features:

- Advanced Vehicle Search: Enable users to search for vehicles based on make, model, price range, mileage, year, fuel type, gearbox, and canton.
- Vehicle Listing Display: Showcase vehicles in an elegant card format with a main photo, price in CHF, key features, and a canton badge.
- Detailed Vehicle View: Provide a comprehensive vehicle details page with an image gallery, detailed description, specifications, and direct contact options (WhatsApp/Email).
- Seller Ad Submission: Allow sellers to submit vehicle listings via a step-by-step form with image upload functionality.
- Location-Based Filtering: Ensure the interface is exclusively in French, displays prices in CHF, and includes a dropdown list of the 26 Swiss cantons for location selection.
- User Authentication and Data Storage: Database solution (Supabase or Firebase) for user authentication and storing advertisement data.
- Mock Data Generation: Generate a set of 10 mock vehicle data entries with prices consistent with the current Swiss market to test the interface.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to inspire confidence.
- Background color: Light grey (#F5F5F5) to maintain a clean and professional appearance.
- Accent color: Muted dark blue (#3F51B5) for highlighting important information.
- Body and headline font: 'Inter', a sans-serif for a modern, machined feel.
- Use universally recognizable icons for vehicle characteristics.
- Prioritize a 'Mobile-first' approach to ensure the application is fully responsive and optimized for mobile devices.
- Implement subtle animations for loading and transitions to improve the user experience.
